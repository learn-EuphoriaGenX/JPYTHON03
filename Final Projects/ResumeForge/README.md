Resume Forge is a modern and easy-to-use resume builder web application that allows users to create professional resumes in minutes. It provides live preview, customizable templates, and downloadable PDF resumes.


Features:-

  Easy-to-fill resume form

  Live preview while editing

  Multiple professional templates

  Download resume as PDF

  Save resume data locally

  Fully responsive design

  Fast and lightweight


Built With:-

    HTML

    Tailwind CSS

    JavaScript

    Local Storage API


Future Improvements:-

    User authentication

    Cloud resume storage

    Multi-language support

    Skill-based resume scoring
